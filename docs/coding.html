CabinetQuote

Cabinetquote was a web-based CRUD application I built back in 2015. In other words, it was a fancy website. The idea was that a homeowner could post a cabinet project that needed to get done. Then a contractor would provide a quote. This was a hobby project and portfolio piece, although I entertained thoughts of actually turning it into a business for a while.

Tech stack:

  • Linux
  • Apache
  • MySQL
  • PHP
  • Bootstrap CSS

Features and functionality

Authentication

A user creates an account linked to an email address.

Content creation

A homeowner posts a project with relevant details, including links to project inspiration on Pinterest.

Reading & filtering

Users can filter for projects in a specific country, state, and city.

Pagination

The site limits the number of results on each page so the content doesn’t get overwhelming.

Updating

A homeowner can change project details anytime.

Messaging

Homeowners and contractors can message each other about projects.

Deleting

A homeowner can easily delete a project.

Error handling

The app doesn’t break if the user makes a mistake while creating an account, logging in, pathing a project, or sending a message.

Security

QC sanitizes input to thwart hacking attempts like SQL injection.
